How it works:
• Mark ALL: Input image is unique vs all listed records. None will show input as duplicate.
• Select subset: Only chosen records treat input as unique. Un-selected records can still show the input image as a potential duplicate and vice versa.
• Once marked unique, those duplicate entries are permanently removed from future searches for this image pair.
